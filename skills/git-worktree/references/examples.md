# Examples

## Create a Worktree

```bash
bash skills/git-worktree/scripts/worktree-manager.sh create feature-login
```

## List Worktrees

```bash
bash skills/git-worktree/scripts/worktree-manager.sh list
```

## Switch Worktrees

```bash
bash skills/git-worktree/scripts/worktree-manager.sh switch feature-login
```

## Cleanup

```bash
bash skills/git-worktree/scripts/worktree-manager.sh cleanup
```
