# Presentation Format

For each pending todo, present enough information for a quick decision without reprinting the entire file.

## Required Fields

- Issue ID and short title from filename
- Priority/severity
- Tags and dependencies
- Problem statement summary
- Relevant findings or technical details
- Proposed solution / recommended action when present

## Format Template

```
Progress: X/Y completed

---
Issue #NNN: <title from filename>
Severity: <p1/p2/p3 label>
Tags: <tags>
Dependencies: <dependencies or none>

Problem:
<short summary>

Findings:
<key findings or technical details>

Proposed action:
<recommended action or solution summary>

Decision? yes / skip / delete / custom
```
