# Error Handling

## Missing Prerequisites

- Verify `gh` CLI or `cm` tools are available.
- Check permissions and authentication.

## No Changed Files

- Verify review target and base branch.
- Confirm branch comparison is correct.

## Agent Failures

- Log agent name and error.
- Continue with remaining agents.
- Report partial results.

## Todo Creation Failures

- Provide fallback instructions using the todo template.
- Link to ../../skills/file-todos/assets/todo-template.md.

## VCS Command Failures

- Provide troubleshooting guidance.
- Suggest manual verification where needed.
